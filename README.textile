h1. SMS C++ client

h2. Download this example

*If you installed git:  git clone git//github.com/h-t-ren/enrima-ws-gsoap-client.git
*If you haven't installed git: go to the webpage https://github.com/h-t-ren/enrima-ws-gsoap-client and click "zip" icon to download the example 
 
h2. Run the example

g++ -o enrima *.cpp -lsocket -lnsl
./enrima
